The Rust programming language is a blazingly fast and memory-efficient programming language. Unlike C/C++, Rust guarantees that your code is memory-safe and thread-safe. Rust also has a friendly compiler, a great toolset, great documentation, and more.

	if second % 2 == 0 {
		first + second
	} else {
		something(first, second - 1)
	}
}

	let ret = something(2, 5);
	println!("something(2, 5) = {}", ret);
